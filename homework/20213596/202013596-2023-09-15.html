Linux命令初步学习

简介

1.阅读《The Linux Command Line A Complete Introduction》的简介和第一章
2.将一个指定的包文件进行解压到指定的文件中
3.解压包文件
4.删除包文件
5.用浏览器播放视频

命令

实现将一个包文件解压、查看、删除为以下操作：
1. cd workdirectory 进入到个人文件夹（workdirectory代指泛称，并非实际的目录名）
2. tar -xf videoClips 解压包文件（包文件的文件名是videoClips）
3. ls vdieoMP4 查看解压后的文件内容（vdieoMP4为解压后的文件名）
4. rm videoClips

注意：
在删除videoClips包文件时会有提示（“rm: remove write-protected regular file 'videoClips'?”）是否要删除一个受写保护规则的文件，按下键盘的“y”选择确认删除

总结

通过阅读《The Linux Command Line A Complete Introduction》的简介和第一章了解到以下内容：
1.命令行虽然使用相对困难，但可以完成许多图形化操作无法完成的事情。
2.在命令终端输入的命令行与操作系统实现交互是将命令行与程序外壳（shell）进行交互再由外壳程序与操作系统底层进行交互，从而实现对操作系统的各种操作。
3.学习到简单、基本的命令行。
